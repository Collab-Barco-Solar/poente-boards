- Library of the board (tbeam): 
    - in preferences paste the link: https://dl.espressif.com/dl/package_esp32_index.json
    - Search of ESP32 in Board Manager
    - Instal the spressif library

- The Arduino LoRa library utilized is the sandeepmistry arduino LoRa at: https://github.com/sandeepmistry/arduino-LoRa 

- The ads1x15 library used is made by adafruit:
    - Search for ads in the Library Manager and install the adafruit ads1x15 library

- The GPS library is the TinyGPS, wich can be download on the link below:
    - http://arduiniana.org/libraries/tinygpsplus/
